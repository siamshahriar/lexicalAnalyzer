# Lexical Scanner
### Analyzable Tokens Dictionary
* Keywords  : int string return if else while switch
* Identifiers   : Any variable name
* Separators    : "" '' . ;
* Brackets  : () {}
* Operators : + - * / = == != <= < > >=
* Literals  : 3.14 etc
* Numbers   : int + double

### Input Code Syntax
Input code should contain spaces.
valid	: `int x = 1`
Invalid	: `int x=1`
